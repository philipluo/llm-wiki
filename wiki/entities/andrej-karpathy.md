# Andrej Karpathy

## Overview

Andrej Karpathy 是 AI 领域知名研究者，曾任特斯拉 AI 总监、OpenAI 联合创始成员。2026 年 4 月发布了 "LLM Wiki" 理念文档，提出用 LLM Agent 持续维护个人知识库的模式。

## Key Facts

- 前特斯拉 AI 总监、OpenAI 联合创始成员
- 热衷于 AI 教育和开源项目（如 nanoGPT、minbpe 等）
- 2026-04-02 发布 LLM Wiki Gist，大几千 Star

## Key Ideas

- 主张不要把 LLM 当搜索引擎（RAG），而要让它像程序员维护代码一样维护知识库
- 提出三层架构：Raw Sources / Wiki / Schema
- 三个核心操作：Ingest / Query / Lint
- 核心洞察：知识应该复利增长，而不是每次从原始文档重新推导
